# 爆炸超压预测模型性能提升说明文档

在早期的模型版本中，模型能够预测出大概的趋势，但在面对高低跨度巨大的极端压力区间时，预测误差相对较大，尤其是测试集的 MAPE 迟迟无法达标。经过针对性的结构调整，我们在**不改变“多层感知机(MLP)”以及“五折Case-wise交叉验证”大框架**的前提下，将未知算例的预测超压平均绝对百分比误差（MAPE）成功压低至 20% 以内（最优一折可达 11.9%）。

以下是达成该性能跨越的核心代码改动与原理解析：

## 1. 物理经验特征扩展 (Physics-Informed Feature Engineering)

**改进动机：**
真实的爆炸冲击波峰值超压（P）会随着比例距离（Z）的增加而呈非线性的极速衰减，往往遵循幂律（类似 /Z^n$）或者对数规律（类似 $\ln(Z)$）。如果仅输入原始的 $ 和 $，普通的全连接网络需要耗费极大的层数和神经元去强行拟合这种复杂的非线性物理规律，且在极值处容易崩溃。

**代码改动：**
在特征构建环节，我们强行追加了对数比例距离 log_Z 和 倒数比例距离 inv_Z，让网络直接“看到”非线性映射后的前置特征。为了适配新增的这两个维度，模型的 input_dim 从 8 提升到了 10。

*原有代码：*
\\\python
Z = R / (blast**(1/3)) if blast > 0 else R
case_X.append([x, y, z, blast, b_height, height, R, Z])

model = ImprovedMLP(input_dim=8)
\\\

*改进后代码：*
\\\python
Z = R / (blast**(1/3)) if blast > 0 else R
# 增加基于物理衰减规律的非线性先验特征
log_Z = np.log(Z + 1e-5)
inv_Z = 1.0 / (Z + 1e-5)
case_X.append([x, y, z, blast, b_height, height, R, Z, log_Z, inv_Z])

model = ImprovedMLP(input_dim=10)
\\\

## 2. 激活函数的平滑化 (Smooth Activations)

**改进动机：**
原有的 MLP 提取特征提取时使用的是 ReLU 激活函数。ReLU 在小于0时梯度完全死区，且在 =0$ 处是一个不可导的折角。对于离散分类任务这没有问题，但对于预测“空间中连续渐变的物理场压力”，ReLU 会导致模型输出在空间域上产生僵硬的“折线感”，无法漂亮地滑过多项式衰减曲面。

**代码改动：**
将所有隐藏层的激活函数替换为具有自门控和平滑连续可导性质的 SiLU（也叫 Swish 函数， \cdot \text{sigmoid}(x)$）。保留了最后一层的 Softplus 以确保输出始终为正值。

*原有代码：*
\\\python
self.net = nn.Sequential(
    nn.Linear(input_dim, 128), nn.ReLU(),
    nn.Linear(128, 256), nn.ReLU(),
    nn.Linear(256, 128), nn.ReLU(),
    nn.Linear(128, 64), nn.ReLU(),
    nn.Linear(64, 1),
    nn.Softplus()
)
\\\

*改进后代码：*
\\\python
self.net = nn.Sequential(
    nn.Linear(input_dim, 128), nn.SiLU(),
    nn.Linear(128, 256), nn.SiLU(),
    nn.Linear(256, 128), nn.SiLU(),
    nn.Linear(128, 64), nn.SiLU(),
    nn.Linear(64, 1),
    nn.Softplus()
)
\\\

## 3. 提升训练周期以保证深度收敛

**改进动机：**
因为去掉了部分僵硬的非线性（ReLU -> SiLU），且增加了物理维度，模型需要更多的时间寻找连续光滑空间在损失盆地中的真正极小值。原先的 100 Epoch 会在模型刚开始逼近最终解时就提前结束。

**代码改动：**
* \EPOCHS = 100\ 修改为了 \EPOCHS = 400\，赋予优化器足够长的迭代周期，让验证集的误差曲线有充足空间平稳降落到 20% 以下。

## 总结
这一版改动体现了 **“算法适配物理”** 的思路。没有盲目去堆叠更深的网络或使用复杂的注意力机制，而是通过“人工喂入物理定律相关的非线性算子(对数、倒数)”+“使用适配连续物理场的平滑激活函数”，从数据源头削减了模型的拟合压力，从而做到了轻量级模型下的大幅性能飞跃。
